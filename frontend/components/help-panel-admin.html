Admin Glossary & Help

Administrative terms for moderation configuration.

Modulation Template
The complete set of rules that determine whether a post is Allowed or Blocked. Templates are versioned and visible to all users per the Visible Modulation principle.
PII (Personally Identifiable Information)
Data that could identify an individual: emails, phone numbers, addresses, full names, etc. The system can block, redact, or flag posts containing PII.
Prompt Injection
Attempts to manipulate the LLM adjudication system through hidden instructions, markdown tricks, or jailbreak prompts. Detection patterns block these attempts.
Snapshot Audit
Immutable record of moderation outcomes per snapshot, including template version, allowed/blocked counts, and block reasons histogram.
Block Reason Code
Standardized codes (e.g., OFF_TOPIC, PII_DETECTED) assigned to blocked posts for transparency and audit purposes.
Template Versioning
All template changes are tracked with semantic versions. New versions apply to future snapshots; existing snapshots remain unchanged.
