What This Page Controls

Use this screen to challenge a stored outcome without changing the live verdict directly. Each filing records the reason for reconsideration, the requested relief, and any evidence references governance reviewers should check.

Appeals stay anchored to the currently selected debate context. Use Snapshot, Audits, and Verdict to inspect the underlying state before submitting a challenge.

Recommended Flow

  1. Review the active snapshot, verdict, and audits before filing.
  2. State the grounds in plain language and be specific about what should be reconsidered.
  3. Name the exact relief you want, such as a re-review or recomputation.
  4. Attach evidence references only when they materially support the appeal.
