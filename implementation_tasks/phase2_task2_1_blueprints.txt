MIGRATION STEPS
───────────────
  1. Create directory structure:
       mkdir -p backend/routes backend/utils
       touch backend/routes/__init__.py backend/utils/__init__.py

  2. Create backend/utils/decorators.py:
       Extract @auth_required, @admin_required, @moderation_check, etc.

  3. Create backend/utils/validators.py:
       Extract validate_string(), validate_json(), validate_debate_input(), etc.

  4. Create backend/utils/serializers.py:
       Extract JSON response wrappers, error response formatters

  5. Create backend/utils/middleware.py:
       • before_request: generate request_id, attach to g
       • after_request: log request/response with request_id
       • error handlers: 404, 500, 429, etc.

  6. Create each blueprint file:
       • Copy relevant routes from app_v3.py
       • Update imports (from backend.utils.decorators import auth_required)
       • Update route decorators (@auth_bp.route(...) instead of @app.route(...))
       • Update url_prefix in blueprint registration

  7. Refactor app_v3.py to app factory pattern:

       from flask import Flask
       from backend.routes import auth_bp, debate_bp, snapshot_bp, governance_bp, admin_bp

       def create_app(config_name="default"):
           app = Flask(__name__)
           app.config.from_object(f"backend.config.{config_name}")

           # Extensions
           from flask_cors import CORS
           CORS(app, ...)  # configured per Task 1.2

           from flask_talisman import Talisman
           Talisman(app, ...)  # configured per Task 1.3

           # Blueprints
           app.register_blueprint(auth_bp, url_prefix="/api/auth")
           app.register_blueprint(debate_bp, url_prefix="/api/debates")
           app.register_blueprint(snapshot_bp, url_prefix="/api/snapshots")
           app.register_blueprint(governance_bp, url_prefix="/api/governance")
           app.register_blueprint(admin_bp, url_prefix="/api/admin")

           # Middleware
           from backend.utils.middleware import setup_middleware
           setup_middleware(app)

           return app

  8. Update all imports in tests, scripts, and wsgi entry points

  9. Verify: python -c "from backend.app_v3 import create_app; app = create_app()"
