Important hidden dependency:
Moving a route to a new Flask blueprint changes its endpoint name even when the
URL is unchanged. Examples:
- debate.submit_post -> posts.submit_post
- debate.get_posts -> posts.get_posts
- debate.list_frame_petitions -> frame_petition.list_frame_petitions
- debate.create_frame_petition -> frame_petition.create_frame_petition

Endpoint names can be used outside route decorators by url_for(), rate-limit
maps, logging, metrics, tests, or helper code. Treat endpoint-name changes as a
real compatibility surface. Public URLs must stay the same; internal endpoint
name references must be found and updated deliberately.
