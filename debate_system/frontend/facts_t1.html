Glossary & Help

Key terms and metrics used throughout the system.

P(true)
Probability that a canonical FACT node is true, as determined by agentic fact-checking. Ranges 0-1; values near 0.5 indicate high uncertainty.
Canonical FACT
An atomic, empirically checkable claim extracted from posts and deduplicated. Forms the evidentiary basis for arguments.
Provenance
Traceable links back to source spans and posts. Shows exactly where each fact originated in the raw text.
Jaccard
Similarity metric for comparing sets. Used here to measure overlap between FACT sets across extraction reruns.
Extraction Stability
How consistently FACT nodes are extracted across multiple pipeline runs. Higher p50 Jaccard = more stable.
